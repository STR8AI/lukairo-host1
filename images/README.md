# Images Directory

This directory contains image assets for the LUKAIRO globe visualization.

## Supported Images

- Earth textures (e.g., `earth.jpg`, `earth.png`)
- Custom map textures
- Any other visual assets

## Usage

Place your texture images here. The build process will optimize them automatically.

To use an Earth texture on the globe, add an image named `earth.jpg` or `earth.png` to this directory.
